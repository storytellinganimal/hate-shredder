# Hate Shredder

A web app that helps people communicate better. You paste a message written in anger or a bad tone, it gets "shredded," and a better-worded version is generated by Codex via the Anthropic API.

## Stack
- React + Vite (single-page app)
- Netlify for hosting + serverless functions
- Anthropic API (Codex-sonnet-4-6) via a Netlify Function that holds the key server-side

## File structure
```
src/
  HateShredder.jsx   — the entire app (UI + prompt logic)
  App.jsx            — mounts HateShredder
  index.css          — minimal reset
netlify/
  functions/
    rewrite.mjs      — serverless proxy to Anthropic API
netlify.toml         — build + functions config
.env                 — local secret (ANTHROPIC_API_KEY), never committed
```

## How it works
1. User types a hostile/angry message
2. Clicks one of three mode buttons — Direct, Candid, or Dialectic
3. Paper-shredder animation plays while the text is sent to `/api/rewrite`
4. `rewrite.mjs` forwards it to Anthropic with a mode-specific prompt
5. Rewritten message appears below

## Three modes
- **Direct** — NVC + Carnegie + Stoic restraint: strips hostility, keeps the point and firmness
- **Candid** — honest emotion with composure: vulnerable but not needy, works in professional and personal contexts
- **Dialectic** — Socratic + principle of charity: steelmans the other side, turns accusations into genuine questions, opens a conversation

## Prompt foundations
All rewrites share a common foundation prompt, then each mode adds its specific instruction. Grounded in:
- Nonviolent Communication (Rosenberg)
- How to Win Friends and Influence People (Carnegie)
- The Righteous Mind (Haidt)
- Aristotle's Rhetoric — ethos, pathos, logos, kairos
- Socratic dialectic and the elenchus
- Seneca / Epictetus on Stoic restraint
- Principle of charity / steelmanning

## Languages
UI and output fully support EN / ES / DE. Language is passed to the API so the rewrite always comes back in the selected language.

## Design rules
- Strictly black and white. No colour, no textures, no gradients.
- 1px borders only. System fonts. Minimal.
- Design is intentionally secondary to content and function.

## Deployment
- Auto-deploys on every push to `main` via Netlify
- Build: `npm run build` → `dist/`
- `ANTHROPIC_API_KEY` lives as an environment variable in Netlify (Site config → Environment variables)
- Never put the key in code or commit it
